## :desktop_computer: 개발 환경
- 이슈 & 형상관리 : Github
- 커뮤니케이션 : Mattermost
- 디자인 : Figma
- 개발 환경
  - DBMS : MYSQL
  - Server : AWS EC2, Ubuntu, nginx
- Front-end
  - React, React-Redux, Yarn
  - Redux-Saga, React-Query
- Back-end
  - Java 8
  - Spring Boot, Spring JPA
- IDE & Tool
  - intelliJ
  - Visual Studio Code
